<h2> Main Features</h2>
<h3>1. Books Management</h3>
<ul>
  <li>Add a book</li>
  <li>Remove a book</li>
  <li>Search by Author or ID</li>
  <li>Display all available books</li>
  <li>Issue and return books</li>
  <li>Check book status (Available or Damaged)</li>
  <li>Show all issued books</li>
  <li>Add new books</li>
  <li>Update quantity</li>
</ul>

<h3>2. Study Room Management</h3>
<ul>
  <li>Show available study rooms</li>
  <li>Reserve a study room</li>
  <li>Cancel reservation</li>
  <li>Assign time slots</li>
  <li>Check reservation status</li>
</ul>

<h3>3. Membership Management</h3>
<ul>
  <li>Display all members</li>
  <li>Register a new member</li>
  <li>Search member by ID or name</li>
  <li>View issued books by a member</li>
  <li>Remove a member</li>
  <li>Update member details</li>
</ul>
